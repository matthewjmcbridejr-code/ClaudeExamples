# LifeOS Static Prototype

This is a tiny no-dependency static prototype generated from the LifeOS Claude Build System idea.

## What it shows

- household members
- today's tasks
- shared calendar items
- shared asset bookings
- e-paper display preview

## How to open

From the repo root:

```bash
xdg-open prototype/lifeos-static/index.html
```

Or double-click `index.html` in the file manager.

## Demo positioning

This prototype is not the finished LifeOS app. It is a simple visual output that shows how the Builder role can turn project instructions into a tangible starting point.

## Safe boundary

- no API keys
- no external dependencies
- no package install
- no private data
- no backend
- no production deployment

## Next step

In a real client setup, the static prototype would become the first visual artifact in the LifeOS project foundation, then the client could approve direction before deeper app development begins.
