# LifeOS User Roles

## Household Owner / Admin

The person who controls household setup.

Can:

- invite and remove members
- assign tasks
- manage shared assets
- set permissions
- configure what appears on the e-paper display

## Adult Member

A trusted household member.

Can:

- view assigned tasks
- create or request tasks
- view shared calendar items
- book shared assets where allowed
- complete tasks

## Child / Dependent

A limited-access household member.

Can:

- view their own tasks
- mark assigned tasks complete
- see approved calendar/display information

Should not be able to:

- edit household settings
- view private adult-only calendar items
- manage other users

## Guest / Helper

A temporary or limited user such as a nanny, driver, assistant, or house staff member.

Can:

- view tasks assigned to them
- update completion status
- see only relevant schedule items

Should not be able to:

- view private household data
- access unrelated shared assets
- edit household membership
