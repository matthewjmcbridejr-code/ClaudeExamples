# LifeOS Feature Map

## Task Management

- create household tasks
- assign tasks to members
- set due dates
- support recurring tasks
- mark tasks complete
- show overdue tasks
- show today's tasks

## Calendar

- show shared household events
- show member-specific events where permitted
- summarize today's calendar
- prepare display-safe calendar output

## Shared Assets

- define shared assets
- view availability
- request or create bookings
- prevent booking conflicts
- apply role-based booking permissions

## Permissions

- household owner/admin
- adult member
- child/dependent
- guest/helper
- display privacy rules

## Notifications

- task due reminders
- overdue reminders
- booking reminders
- daily summary

## E-Paper Display

- today's tasks
- assigned owners
- top calendar items
- current date/time
- next refresh time
- privacy-filtered display mode

## Future Automation

- suggest recurring tasks
- detect schedule conflicts
- produce family planning summaries
- connect approved MCP tools
