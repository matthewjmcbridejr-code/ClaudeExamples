# LifeOS MVP Roadmap

## Phase 1 — Tasks + Calendar Foundation

Goal: make the household's daily work visible and assignable.

Includes:

- household setup
- household members
- task creation
- task assignment
- due dates
- recurring tasks
- daily task view
- simple calendar summary

Success condition:

A family can see who is responsible for what today.

## Phase 2 — Shared Assets + Bookings

Goal: manage family resources without confusion.

Includes:

- shared asset list
- booking calendar
- basic availability rules
- owner/admin approval settings
- conflict prevention

Success condition:

A family can reserve shared items or spaces without double-booking.

## Phase 3 — E-Paper Display

Goal: create a low-distraction home dashboard.

Includes:

- display-ready endpoint
- today's tasks
- assigned owners
- current calendar highlights
- refresh cadence
- privacy filters

Success condition:

The home display shows only useful, approved, current information.

## Phase 4 — Automation / Agents

Goal: add intelligence after the core workflows are proven.

Potential additions:

- task suggestions
- overdue task reminders
- schedule conflict detection
- family planning assistant
- controlled MCP tools for calendar, docs, tasks, and deployment

Success condition:

Automation saves time without creating noise or privacy risk.
